HTML & CSS Quick Reference Guide

A quick guide for commonly used HTML tags and CSS properties.

HTML Tags

<h1> - <h6>: Headings Tag

Defines HTML headings from the largest (<h1>) to the smallest (<h6>).

<h1>This is a heading</h1>

<p>: Paragraph Tag

Defines a block of text as a paragraph.

<p>This is a paragraph.</p>

<b>: Bold Tag

Makes text bold.

<b>This text is bold</b>

<i>: Italic Tag

Makes text italic.

<i>This text is italic</i>

<u>: Underline Tag

Underlines text.

<u>This text is underlined</u>

<br>: Line Break

Inserts a line break in the text.

This is line 1.<br>This is line 2.

<img>: Image Tag

Embeds an image in the webpage.

<img src="image.jpg" alt="Description">

<ul>, <ol>, <li>: Lists

Defines unordered (<ul>) and ordered (<ol>) lists with list items (<li>).

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<form>: Form Tag

Creates a form for user input.

<form action="submit.php" method="post">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    <br>
    <input type="submit" value="Submit">
</form>

CSS Properties

color

Sets the text color.

p {
    color: blue;
}

font-family

Specifies the font style of the text.

body {
    font-family: Arial, sans-serif;
}

font-size

Specifies the size of the text.

h1 {
    font-size: 24px;
}

margin and padding

Defines the spacing around an element.

div {
    margin: 20px;
    padding: 10px;
}

border-bottom

Applies a bottom border to an element.

h1 {
    border-bottom: 2px solid black;
}

side-borders (border-left, border-right)

Defines borders on the left and right sides of an element.

div {
    border-left: 5px solid red;
    border-right: 5px solid blue;
}

background-color

Defines the background color of an element.

body {
    background-color: #f0f0f0;
}
